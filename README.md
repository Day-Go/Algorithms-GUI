# Algorithms-GUI
A desktop GUI for creating animations of algorithms. The tool generates high-quality animations using the [Manim package](https://github.com/ManimCommunity/manim).

![alt text](https://github.com/OolongJunSun/Algorithms-GUI/blob/main/demo/BubbleSort_color_node.gif?raw=true)
![alt text](https://github.com/OolongJunSun/Algorithms-GUI/blob/main/demo/BubbleSort_long_1080p.gif?raw=true)

## Features
- Generate sequences automatically within pre-defined ranges.
- Select task and algorithm.
- Configure output video parameters and visualization style

## Supported tasks & algorithms
- Sorting
  - [x] Bubble sort
  - [ ] Selection sort
  - [ ] Insertion sort
  - [ ] Merge sort
